## Why this module (SEO summary)
- Improve conversions by allowing customers to pick configurable options directly from the wishlist page.
- Reduces friction and product page visits — ideal for mobile shoppers.
- Hyvä Themes compatible: designed to integrate with Hyvä frontends for faster, modern storefronts.

## Key Features
- Select swatches (color, size, pattern) for configurable products from the wishlist.
- AJAX-based updates: instant selection updates without page reloads.
- Mobile-first, responsive UI designed for performance.
- Compatible with Magento Open Source & Commerce (2.3.x, 2.4.x) and Hyvä Themes.
- Lightweight and easy to integrate with custom themes and third-party wishlist extensions.

## Installation
1. Copy the module into the Magento codebase:

```bash
cp -R MyWishlistWithSwatches app/code/AumTechnolabs/MyWishlistWithSwatches
```

2. Enable and run upgrade/compile/cache commands:

```bash
php bin/magento module:enable AumTechnolabs_MyWishlistWithSwatches
php bin/magento setup:upgrade
php bin/magento setup:di:compile
php bin/magento cache:flush
```

3. (Production) Deploy static content:

```bash
php bin/magento setup:static-content:deploy -f
```

## Hyvä Themes Integration Notes
- This module ships with template-friendly, theme-agnostic blocks. For Hyvä, the module's frontend templates are compatible out of the box. To ensure proper Hyvä styling:
	- Verify `view/frontend/templates/*` are mapped in your theme or use Hyvä's override patterns.
	- If you use Hyvä's Tailwind setup, add minimal utilities or small CSS where needed to match your Hyvä design system.
	- The `view/frontend/requirejs-config.js` and `view/frontend/web/js/wishlist.js` work with Hyvä's client-side setup — you may optionally move the script to Hyvä's JS pipeline if you bundle assets.
